深度优先搜索（DFS）是遍历树、图、状态空间的基础算法，核心是**一条路走到底，走不通就回溯，换条路继续探索**，天然适配递归与回溯思想，是全排列、组合、迷宫、N 皇后等问题的通用解法。

## 一、DFS 到底是什么？

一句话概括：**从起点出发，优先往深处探索，直到无法前进，再回退到上一分支，遍历所有可能路径**。

通俗理解：走迷宫时，遇到岔路就选一条一直走，走到死胡同就原路返回，换另一条没走过的路，直到走遍所有通道或找到出口。

### 核心三要素

1.  **深度优先**：不横向扫，只纵向挖，不撞南墙不回头。
2.  **回溯**：走到尽头 / 不满足条件时，撤销上一步选择，恢复状态，尝试其他分支。
3.  **状态标记**：用数组 / 哈希表记录已访问节点，避免重复遍历与死循环。

## 二、DFS 的执行流程（递归版最易懂）

以**1~n 全排列**为例，直观拆解每一步：

1.  确定当前要填第 u 位数字；
2.  枚举 1~n 中未使用的数字；
3.  标记该数字已用，填入当前位；
4.  递归处理下一位 u+1；
5.  **回溯**：取消标记，恢复状态，继续枚举其他数字；
6.  当 u>n 时，输出完整排列。

## 三、经典例题：1~n 全排列（C++ 代码）

```cpp

#include <iostream>  
using namespace std;  
  
const int N = 10;  
int path\[N\];    // 存储当前排列  
bool state\[N\];  // 标记数字是否已使用  
int n;  
  
// u：当前正在填第几位  
void dfs(int u) {  
    // 递归出口：已填完所有位，输出结果  
    if (u > n) {  
        for (int i = 1; i <= n; ++i)  
            cout << path\[i\] << " ";  
        cout << endl;  
        return;  
    }  
    // 枚举所有可选数字  
    for (int i = 1; i <= n; ++i) {  
        if (!state\[i\]) {  
            path\[u\] = i;       // 选择i  
            state\[i\] = true;  // 标记已用  
            dfs(u + 1);       // 递归下一位  
            state\[i\] = false; // 回溯：撤销选择  
        }  
    }  
}  
  
int main() {  
    cin >> n;  
    dfs(1); // 从第1位开始  
    return 0;  
}  

```

### 代码关键点

-   **path 数组***：记录当前路径 / 方案；
-   **state 数组**：防止重复选择，保证排列合法；
-   **递归 + 回溯**：选→递归→撤销，是 DFS 的灵魂模板。

## 四、DFS 的两种实现方式

### 1\. 递归实现（推荐）

-   优点：代码极简、逻辑直观，贴合回溯思维；
-   缺点：深度过大可能栈溢出（如 n>1e4）；
-   本质：用系统函数栈自动维护路径与回溯。

### 2\. 非递归实现（手动栈）

用`stack`存节点 / 状态，循环模拟 “入栈 — 访问 — 出栈 — 回溯”，适合大数据、避免栈溢出。

## 五、DFS 适用场景

-   ****组合枚举****：全排列、子集、组合数；
-   ****路径搜索****：迷宫、两点是否连通、找任意一条路径；
-   ****回溯问题****：N 皇后、数独、括号生成；
-   ****图 / 树遍历****：连通分量、拓扑排序、判环、前 / 中 / 后序遍历。
